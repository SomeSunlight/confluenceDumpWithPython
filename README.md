# 🚀 Project Moved to ConfluenceBridge

This repository is **deprecated** and no longer maintained. 
The project has been completely refactored, modernized, and moved to a new standalone repository:

## 👉 [**ConfluenceBridge**](https://github.com/SomeSunlight/ConfluenceBridge)

### Why switch to ConfluenceBridge?
* **Modern ETL Architecture:** Stable data pipeline for extraction and transformation.
* **Delta-Sync:** Significantly faster subsequent downloads using `manifest.json`.
* **Playwright Fallback:** Robust rendering for complex Confluence macros/tables.
* **LLM-Ready:** Optimized output for AI ingestion and offline documentation.
